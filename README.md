# ESP-WROOM-02-RGB
<p>ESP-WROOM-02を使ってウエブブラウザからRGB-LEDを制御します
<p>切り替えスイッチでステーションモード／アクセスポイントモードを選択

<p>アクセスポイントモード時
<p>SSID:ESP8266-RGB PW:12345678でアクセスポイントに接続
<p>ブラウザで http://192.168.1.4 を開く
<p>
<p>ステーションモード時
<p>既存のアクセスポイントのSSIDとPWをssid.txt、passwd.txtと記述して
<p>SDメモリに保存
<p>起動後液晶に表示されるアドレスをブラウザで開く
<p>（アドレスはアクセスポイントのDHCPにより割り当てられる）
<p>
<p>制御ページのHTMLデータはSDメモリに格納
